  2. Utilisez une boucle for pour parcourir la liste.
  3. Dans la boucle, utilisez un opérateur de comparaison pour vérifier si le nombre est divisible par 3 (utilisez l'opérateur modulo %).

# Entrez votre code ici pour manipuler les structures de contrôle. # Solution de l'exercice nombres = list(range(1, 16)) print("Nombres divisibles par 3:") for nombre in nombres: if nombre % 3 == 0: print(f"{nombre} est divisible par 3") print("\nNombres divisibles par 3 et >= 10:") for nombre in nombres: if (nombre % 3 == 0) and (nombre >= 10): print(f"{nombre} est divisible par 3 et supérieur ou égal à 10") print("\nAffichage des cinq premiers nombres avec boucle while:") i = 0 while i < len(nombres): print(nombres[i]) i += 1 if i == 5: break test_function("print") success_msg("Félicitations ! Vous avez réussi à manipuler les structures de contrôle en Python.")

Exercice 5: Structures séquentielles

Instructions :

Écrivez un script en Python qui réalise les tâches suivantes :

  1. Création et manipulation de liste:
    • Créez une liste fruits contenant les éléments "pomme", "banane" et "cerise".
    • Ajoutez "orange" à la fin de la liste.
    • Supprimez "banane" de la liste.
    • Affichez chaque élément de la liste en utilisant une boucle for.
  2. Manipulation de dictionnaire:
    • Créez un dictionnaire personne avec les clés "nom" et "age", contenant respectivement "Alice" et 30.
    • Ajoutez la clé "ville" avec la valeur "Paris".
    • Modifiez la valeur de "age" à 31.
    • Affichez toutes les paires clé-valeur en utilisant une boucle for.
  3. Utilisation des ensembles:
    • Créez un ensemble ensemble1 contenant les nombres 1, 2, 3 et 4.
    • Créez un ensemble ensemble2 contenant les nombres 3, 4, 5 et 6.
    • Affichez l'intersection et l'union de ces deux ensembles.
  4. Manipulation de tuple:
    • Créez un tuple dimensions avec les valeurs (10, 20, 30).
    • Affichez chaque élément du tuple en utilisant une boucle for.
# Entrez votre code ici pour manipuler les structures séquentielles. # Solution de l'exercice fruits = ["pomme", "banane", "cerise"] fruits.append("orange") fruits.remove("banane") print("Liste des fruits:") for fruit in fruits: print(fruit) personne = {"nom": "Alice", "age": 30} personne["ville"] = "Paris" personne["age"] = 31 print("\nDétails de la personne:") for cle, valeur in personne.items(): print(f"{cle}: {valeur}") ensemble1 = {1, 2, 3, 4} ensemble2 = {3, 4, 5, 6} print("\nIntersection:", ensemble1 & ensemble2) print("Union:", ensemble1 | ensemble2) dimensions = (10, 20, 30) print("\nDimensions du tuple:") for dimension in dimensions: print(dimension) test_function("print") success_msg("Bravo ! Vous avez maîtrisé les structures séquentielles en Python.")
Pratiquez la manipulation des listes, des dictionnaires, des ensembles et des tuples.
